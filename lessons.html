Learning Laboratory

Master hardware fundamentals through structured modules with clear learning objectives and practical applications.

01. Core Electronics

Beginner 15 MIN

Ohm's Law Masterclass

Learn the fundamental relationship V = I × R. Calculate current flow, voltage drops, and power dissipation in any circuit.

Key Learning Objectives:

  • Calculate resistance needed for LED circuits
  • Solve series and parallel resistor networks
  • Understand power ratings (watts = volts × amps)
  • Prevent component damage from overcurrent
Beginner 20 MIN

Component Anatomy

Identify electronic components by sight and symbol. Read resistor color codes and capacitor markings accurately.

Key Learning Objectives:

  • Decode 4/5-band resistor color codes
  • Read ceramic/electrolytic capacitor values
  • Identify diode markings and LED polarity
  • Test components with multimeter
Beginner 25 MIN

Power Supply Basics

Build stable 5V/3.3V power supplies. Understand voltage regulation and filtering.

Key Learning Objectives:

  • Use 7805 regulator for 5V output
  • Add capacitors for ripple filtering
  • Calculate heat sink requirements
  • USB vs barrel jack power options
Intermediate 35 MIN

Kirchhoff's Laws

Solve complex circuits using KVL (voltage) and KCL (current) laws.

Key Learning Objectives:

  • Apply KVL around circuit loops
  • Use KCL at circuit junctions
  • Solve voltage divider circuits
  • Analyze current sharing in parallel

02. Arduino & Logic

Intermediate 45 MIN

The Setup() & Loop()

Master Arduino program structure. Write efficient, readable code.

Key Learning Objectives:

  • Initialize pins in setup()
  • Use loop() for continuous operation
  • Declare global vs local variables
  • Avoid common blocking code errors
Intermediate 60 MIN

Digital vs Analog

Understand binary (digital) vs continuous (analog) signals.

Key Learning Objectives:

  • digitalRead() / digitalWrite() usage
  • analogRead() returns 0-1023
  • PWM with analogWrite() on pins 3,5,6,9,10,11
  • Map sensor values to outputs
Advanced 90 MIN

I2C Communication

Connect multiple devices using 2-wire I2C protocol.

Key Learning Objectives:

  • Wire.begin() and device addresses (0x68 MPU6050)
  • Read/Write registers with Wire library
  • I2C scanner to find connected devices
  • Handle multi-byte transfers
Advanced 75 MIN

SPI Protocol Mastery

High-speed 4-wire communication for displays and sensors.

Key Learning Objectives:

  • SPI pins: 11(MOSI), 12(MISO), 13(SCK)
  • Chip Select (CS) for multiple devices
  • SPI clock modes 0,1,2,3 configuration
  • High-speed SD card and OLED display
Advanced 120 MIN

Interrupts & Timers

Replace polling with event-driven interrupts.

Key Learning Objectives:

  • attachInterrupt() on pins 2,3
  • Timer1/2 for precise timing
  • ISR (Interrupt Service Routine) best practices
  • Non-blocking code architecture
